# University Student Data Dashboard

## Purpose
This project analyzes university student data related to applications, admissions, enrollment, retention, satisfaction, and departmental enrollment.  
It was developed for the activity **Data Visualization and Dashboard Deployment**.

## Project Files
- `app.py`: Streamlit dashboard application.
- `requirements.txt`: Python libraries required to run the app.
- `university_student_data.csv`: Dataset used in the analysis.
- `README.md`: Project description and execution instructions.

## Main Dashboard Features
- Interactive filters by year, term, and department.
- KPI cards for applications, admitted students, enrolled students, average retention, and average satisfaction.
- Line chart for retention rate trends over time.
- Bar chart for student satisfaction by year.
- Grouped bar chart for Spring vs. Fall comparison.
- Donut chart for enrollment distribution by department.

## How to Run Locally
1. Clone this repository.
2. Open a terminal in the project folder.
3. Install the dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
streamlit run app.py
```
